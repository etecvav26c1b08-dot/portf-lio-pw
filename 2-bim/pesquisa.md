# Definiçao de css
#### CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para definir a aparência visual de documentos HTML. Ela separa o conteúdo (estrutura) do design (apresentação), controlando cores, fontes, layouts, espaçamentos e responsividade, permitindo que sites sejam estilizados e adaptados para diferentes tamanhos de tela. 

Principais características e usos:
- Separação de Estrutura e Estilo: O HTML define o conteúdo e o CSS define como ele é exibido.
- Design Responsivo: Essencial para criar sites que se ajustam automaticamente a celulares, tablets e computadores.
- Reutilização de Código: Um único arquivo CSS pode estilizar várias páginas de um site.
- Propriedades de Estilização: Define cores de fundo, tipos de fonte, margens, alinhamentos, bordas e animações.
- Padrão W3C: É uma tecnologia padronizada para web, evoluindo do CSS2 ao CSS3 (em módulos).
